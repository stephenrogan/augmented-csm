# Report Templates

## Weekly CS Leadership Report
1. Portfolio health summary (1 paragraph)
2. Outlier segments (table)
3. Top movers (5 accounts with largest health changes)
4. Expansion pipeline summary
5. Action items
Length: 1 page maximum.

## Monthly Board/Investor Report
1. NRR by segment (chart)
2. Health distribution trend (chart)
3. Churn analysis
4. Expansion analysis
5. Outlook
Length: 2-3 pages with charts.
