# Stakeholder Classification Guide

## Types
- Champion: actively advocates internally
- Economic Buyer: controls or influences budget
- Technical Lead: owns implementation
- End User: uses the product daily
- Executive Sponsor: provides air cover
- Influencer: shapes opinions (positive or negative)
- Detractor: works against the product/relationship

## Coverage Status
- Active: engaged within expected cadence
- Cooling: 1.5-2x expected cadence since last touch
- Gap: >2x expected cadence
- Lost: no engagement in 90+ days

## Classification Responsibility
CSM classifies contacts. The agent tracks engagement. Classification should be reviewed at every QBR.
