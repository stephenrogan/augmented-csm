# Feedback Taxonomy

## Categories
- Feature gaps: capabilities the product does not have
- Feature improvements: existing features that need enhancement
- UX/usability: the feature exists but is hard to use
- Performance: speed, reliability, scale
- Integration: connections to other tools

## Ranking Criteria
1. Unique accounts requesting (breadth)
2. ARR weight of requesting accounts (revenue impact)
3. Correlation with churn or health decline (retention impact)

## Deduplication Rules
Multiple tickets from the same account on the same topic count as one request with severity weight, not multiple requests.
