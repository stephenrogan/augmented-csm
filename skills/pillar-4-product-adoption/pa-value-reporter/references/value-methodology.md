# Value Measurement Methodology

## Universal Metrics
- Time saved: automated workflow completions * estimated manual time per workflow
- Productivity gain: output per user, current vs. baseline period
- Adoption ROI: features adopted * estimated value per feature

## Custom Metrics
Defined per account during onboarding or QBR. Must be measurable and customer-validated.

## Estimation Principles
- Conservative estimates over impressive ones. Credibility matters more than size
- Always frame as "estimated" -- never present as precise
- Validate with the customer before using in renewal or executive conversations
