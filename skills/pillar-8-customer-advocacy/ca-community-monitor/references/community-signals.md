# Community Signal Definitions

| Signal | Detection | Action |
|--------|-----------|--------|
| Unanswered question (48h+) | Post with no reply | Alert CSM or community manager |
| Negative sentiment | Frustrated/critical language | Alert CSM -- public frustration needs attention |
| High-value contributor | 5+ answers per month | Flag as potential advocate |
| New member | Customer contact joins | Log in ra-engagement-tracker |
| Feature discussion thread | Multiple customers discuss a gap | Feed into pa-feedback-aggregator |
| Competitor mention | Evaluation language | Feed into bi-competitive-intel |
